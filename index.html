Stock Price Forecasting

This is a research project on stock price forecasting using Echo State Networks(ESNs). It was implemented in Matlab. I applied the ESN on stock price forecasting and compared its performance with Backpropagation neural networks and Radial basis function networks'. In order to get a better trained model, I then used particle swarm optimization(PSO) to find a set of parameters for generating the ESN.
